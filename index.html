What does javascript event loop do?

Javascript Event loop made by queue,stack and heap.Javascript event loop is responsible for executing code and collecting and processing event and sub-task.Event loops take the event from the queue and places onto the stack.The call stack responsible for function opertion execution and this function calling other functions inside it, if any.If the operation is finished it's popped from the stack.The call back function wait in the event queue and the Web API is making the mentioned wait. Difference between local storage and session storage?


Local Storage

1. Local storage data doesn't expire.

2. The data is persisted until the user manually clears the browser cache or until the web app clears the data.

3. The storage capacity of local storage is 5MB/10MB.

4. Local storage is not session based.

5. window.localStorage

6. The data will not be deleted when the browser is closed

Session Storage

1. Session storage data expire.

2. The data is persisted only until the window or tab is closed.

3. The storage capacity of session storage is 5MB

4. Session storage is session based.

5. window.sessionStorage

6. The data is deleted when the user closes the specific browser tab.
